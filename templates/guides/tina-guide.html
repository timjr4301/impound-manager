The Ohio title rule — when you can file and when you can't

Important — read this
The basic rule

Two requirements must both be met before you can file

  • At least 60 days have passed since the impound date
  • At least 30 days have passed since Letter 2 was sent

Both conditions must be true. The system calculates this automatically and shows the eligible date on each vehicle.

The NADA value rule — critical

Before filing, check whether the vehicle value justifies standard title

Ohio Title-by-Abandonment Rule

NADA Rough Trade-In Value minus total charges owed (tow + storage) must equal a negative number.

Example: NADA value $1,200 — charges owed $3,500 = -$2,300 ✓ Can file standard title

Example: NADA value $8,000 — charges owed $3,500 = +$4,500 ✗ Cannot file standard title — need court order

If the number is positive (vehicle is worth more than what's owed), you cannot file title-by-abandonment. A court order is required. Flag this for Tim.
If NADA shows "—" or $0, the system couldn't look up a value. You need to check NADA manually or have Tim enter a manual value before you file.
PO Box rule

If the owner's address is a PO Box, special rules apply

If the BMV shows the owner's address as a PO Box, the certified letter process has different requirements. The system flags these automatically. If you see a PO Box flag, ask Tim before filing.

Filing a title

Core task
Starting the filing process

Click "File Title" on any eligible vehicle

File for Title — shows the vehicle is eligible, with all dates confirmed. Enter the BMV receipt number and notes.

This page confirms the vehicle is eligible, shows the impound date, days in storage, eligible-since date, and when Letter 2 was sent.

Towbook data on every vehicle

Scroll down on any vehicle record to see Towbook Data

Towbook Data — Stock #, Call #, Invoice #, Account, Balance Due, Have Keys status, and open tasks from Towbook

This section pulls live data from Towbook. The Have Keys field is important — if it says No, Robert needs to make keys before the vehicle can go to auction. The Balance Due shows what's currently owed in Towbook.

Damage items on the BMV form

Add damage items before generating the BMV 4202

Damage Items preset dropdown — select from common presets or type a custom description and dollar amount

The BMV 4202 form allows up to 9 damage items. Use the preset dropdown to quickly add common damage types. The amounts are pre-filled but you can adjust them.

  • KEY REPLACEMENT — $350
  • FRONT BUMPER — $450
  • FENDER — $350
  • HOOD — $500
  • WINDSHIELD — $300
  • WHEEL / TIRE — $300
  • QUARTER PANEL — $600
  • INTERIOR — $300
  • MECHANICAL — $1,200
Max 9 items on the BMV form. If a vehicle has more damage than that, combine similar items or use the highest-value ones.
Generating the BMV 4202

Click "Generate BMV 4202" to create the title packet PDF

The BMV 4202 PDF saves automatically with the VIN and date in the filename — save it to your desktop or Downloads

The PDF is auto-filled with all vehicle info, damage items, fees, and your company information. Save it, print it, and take it to the BMV.

NADA lookup

Click "Look Up NADA Value" to get the wholesale value automatically

NADA lookup — click OK and the system opens Edmunds and pulls the rough trade-in value automatically (takes 1-2 minutes)
Always use the worst-case condition when looking up NADA — flood damage yes, frame damage yes, fire damage no, title status unknown. This gives the lowest defensible wholesale value, which is what Ohio requires for title-by-abandonment.
What to fill in

Enter the BMV receipt number and any notes

Take the vehicle paperwork to the Ohio BMV
Submit the title application (BMV form 4202 for PPI, 4205 for police)
Get the receipt — it has a BMV application number on it
Come back to the system and enter that number in the BMV Receipt / Application # field
Add any notes (drop-off location, agent name, anything relevant)
Click the confirm button — this marks the vehicle as Title Filed and removes it from the active dashboard
Note: Once you click confirm, the vehicle moves to the Title Filed tab and disappears from your active queue. Make sure everything is right before clicking.

Setting vehicle disposition — sell, junk, or hold

Every vehicle needs a decision
What is disposition?

After title is filed, every vehicle needs a decision on what happens to it

Disposition — set SELL, JUNK, or HOLD. The NADA value shows at top to help you decide.

Click Set next to any vehicle in your dashboard to open the disposition panel. The NADA value shows at the top so you know what the vehicle is worth.

Three options

Choose based on the vehicle's condition and value

SELL

Auction or private sale. Vehicle goes into the auction pipeline.

JUNK

Scrap or junk yard. Vehicle gets an invoice for scrap price.

HOLD

Pending decision. Use when you're not sure yet or waiting on Tim.

Tip from Tim: If you're not sure, set HOLD and make a note. Tim reviews vehicles on hold regularly.

Valuation Report

For every vehicle before filing
What it shows

Click "Valuation Report" from any vehicle record

Valuation Report — shows NADA wholesale value, total fees owed, fee breakdown, and disposition decision checkboxes

The Valuation Report shows:

  • NADA Wholesale Value — what the vehicle is worth at wholesale (rough trade-in)
  • Total Amount Owed — tow fee + accumulated storage days × daily rate
  • Fee Breakdown — itemized tow fee and storage calculation
  • Disposition Decision — checkboxes for Sell, Junk, or Hold
The math that matters

NADA value minus total owed tells you if you can file standard title

Quick check before filing

Look at NADA Wholesale Value and Total Amount Owed on this report.

NADA − Total Owed = negative number → File standard title ✓

NADA − Total Owed = positive number → Need court order, flag Tim ✗

You can also print this report for your records before going to the BMV. Click Print Report at the top.
